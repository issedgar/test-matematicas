# Operaciones matemáticas

Módulo para relizar operaciones matematicas simples

## Installation

Using npm:

`npm install ehc-test-matematicas`

## Usage


```ts

// Importar el módulo
const m = require('.');

// Sumas
console.log(m.suma(2, 5)); => 7

// Restas
console.log(m.suma(2, 5)); => -3
```
